Примеры

  • Обработка стат. данных на примере данных испытания бетона
  • Корреляционный анализ
  • Интерполяция
  • Линейное программирование
  • Расчёт балки по предельным состояниям методом линейного программирования

    • Примеры написанны на языке программирования Python и представленны в виде файлов Jupyter Notebook. Запустить примеры можно в программе Anaconda или онлайн, в mybinder.org (запуск прямо по ссылке, examples - папка с примерами).

    Для запуска примера в Anaconda, нужно в Anaconda Navigator выбрать программу Jupyter Notebook. Подождать пока откроется вкладка в браузере со списком файлов домашней папки пользователя, открыть файл с примером. Для непостредсвенного запуска примера в меню Cells выбрать Run All.
